## 基于halcon的光栅投影的三维重构的研究

> 我的那篇论文

地址 => [](https://halcon.inner.ink)

### 介绍

本项目基于halcon这一功能强大的算法包和光栅投影的原理去实现三维重构

### 由来

题目来源来我的本科毕业论文，在我进行研究的过程中发现网上这块的理论知识很多，但是实际操作的完整流程和代码都没有。
在我的努力下终于把这篇论文写完，实验部分也完整的由自己做完，所以想把整个过程都详细的写下来，方便你们学习和研究。
